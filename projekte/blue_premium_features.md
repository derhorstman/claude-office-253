# Premium Features für Blue

## Dokumente
- PDF erstellen (schön formatiert)
- PDF aus Vorlagen generieren
- Word/DOCX Export

## Bilder
- DALL-E 3 Bildgenerierung (OpenAI API)
- Bilder resize/bearbeiten

## Sprache
- Text-to-Speech mit ElevenLabs (Antworten vorlesen)
- Speech-to-Text mit Whisper (Spracheingabe)

## Vorlagen für Pflege
- Abmahnung
- Kündigung
- Arbeitszeugnis
- Gesprächsprotokoll
- MDK-Widerspruch
- Dienstanweisung

## Kalender
- Termine verwalten
- Erinnerungen setzen

## Notizen
- Schnelle Textnotizen
- Sprachnotizen (aufnehmen + transkribieren)

## QR-Codes
- QR-Codes generieren für Links/Infos
